# 스마트 문서 스캐너 PWA

문서를 촬영하거나 사진을 불러와 보정한 뒤 여러 장을 하나의 PDF로 저장·공유하는 PWA 앱입니다.

## 주요 기능

- 카메라 촬영
- 사진 여러 장 불러오기
- 원본 / 선명 / 흑백 / 밝게 / 대비 / 구김완화 보정
- 회전
- 손가락 4점 자르기
- 자동 테두리 추천
- 비스듬히 찍힌 문서 펼치기
- 페이지 순서 변경
- PDF 품질 선택(고화질/일반/저용량)
- 문서 종류 선택 및 자동 파일명
- 최근 문서 검색
- 저장공간 확인 및 오래된 문서 정리
- PDF 생성
- 미리보기 / 저장 / 공유
- 최근 PDF 목록 저장
- PWA 설치 지원

## 실행 방법

로컬에서 바로 확인하려면 폴더에서 아래 명령어를 실행하세요.

```bash
python3 -m http.server 8080
```

그 다음 브라우저에서 아래 주소로 접속합니다.

```text
http://localhost:8080
```

스마트폰 카메라 기능은 HTTPS 환경에서 가장 안정적으로 작동합니다. Vercel, Netlify, GitHub Pages 등에 올리면 HTTPS로 사용할 수 있습니다.

## Vercel 배포 방법

1. 이 폴더 전체를 GitHub 저장소에 업로드합니다.
2. Vercel에서 New Project를 누릅니다.
3. 해당 저장소를 선택합니다.
4. Framework Preset은 Other 또는 Static으로 둡니다.
5. Build Command는 비워둡니다.
6. Output Directory도 비워두거나 `.` 으로 둡니다.
7. Deploy를 누릅니다.

## 주의사항

- PDF 생성에는 jsPDF CDN을 사용합니다.
- 첫 접속 시 인터넷 연결이 필요합니다.
- 한 번 로딩된 뒤에는 앱 파일과 PDF 라이브러리를 캐시에 저장합니다.
- 최근 PDF는 사용자의 브라우저 내부 저장소 IndexedDB에 저장됩니다.


## 이번 수정 내용

- 기존 슬라이더 자르기를 제거하고, 사진 위의 네 모서리 손잡이를 손가락으로 움직이는 방식으로 변경했습니다.
- 네 모서리를 문서 귀퉁이에 맞춘 뒤 `문서 펴서 적용`을 누르면 PDF 생성 시 해당 영역을 반듯한 사각형 문서처럼 보정합니다.
- 구겨진 종이의 그림자와 흐릿함을 완화하는 `구김완화` 보정 버튼을 추가했습니다.


## 85점 버전 추가 내용

- 사진 추가 시 문서 영역을 자동으로 대략 추천하도록 개선했습니다.
- 자르기 화면에 `자동 테두리 추천` 버튼을 추가했습니다.
- PDF 생성 전에 문서 종류와 장소/메모를 선택하면 파일명이 자동으로 만들어집니다.
- PDF 품질을 고화질, 일반, 용량 작게 중에서 고를 수 있습니다.
- 최근 문서에서 파일명, 문서 종류, 날짜로 검색할 수 있습니다.
- 저장된 PDF 개수와 용량을 확인하고, 오래된 문서를 정리할 수 있습니다.


## 2026-04-26 수정사항

- 자동 테두리 추천 알고리즘을 개선했습니다. 단순 사각형 전체 선택이 아니라 문서의 기울어진 네 변을 추정해 네 모서리를 추천합니다.
- 자동 인식 실패 시 사진 끝까지 붙지 않고 약간의 안쪽 여백을 둔 자르기 영역을 표시해 손가락 조정이 쉬워지도록 했습니다.
- 흑백 보정에서 그림자 부분이 까맣게 뭉개지는 문제를 줄이기 위해 로컬 배경 보정 기반의 흑백 처리로 변경했습니다.
- 구김완화 보정도 그림자 완화가 먼저 적용되도록 개선했습니다.


## v5 수정 사항

- 스캔 페이지의 사진 썸네일을 누르면 현재 보정 상태를 크게 확인할 수 있습니다.
- 페이지 카드에 현재 보정 방식, 회전 각도, 문서영역 상태를 표시합니다.
- 흑백 보정에 밝기 슬라이더를 추가했습니다. 배경이 너무 밝으면 70~90% 쪽으로 낮춰 조정할 수 있습니다.
- 기존 '구김완화' 표현을 '그림자 완화'로 변경했습니다. 이 기능은 구겨진 종이를 실제로 펴는 기능이 아니라 그림자·접힘 자국·얼룩을 완화하는 문서 정리 보정입니다.
- 흑백 기본 밝기를 낮춰 배경이 과하게 하얗게 날아가는 현상을 줄였습니다.


## v6 수정 사항
- 흑백 스캔에서 종이 배경이 회색으로 남는 문제를 개선했습니다.
- 배경은 흰색에 가깝게 정리하고 글자와 표 선은 진하게 남기도록 흑백 알고리즘을 변경했습니다.
- 흑백 밝기 조절은 배경 흰색 강도를 조절하는 방식으로 변경했습니다.
